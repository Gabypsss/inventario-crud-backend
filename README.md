#inventario-crud-backend

**Sistema de Inventario – Backend (CRUD en Java)**

Proyecto desarrollado como parte del trabajo en equipo en la Universidad Libre.  
Este repositorio contiene **mi aporte personal al backend**, donde se implementa el CRUD principal del sistema de inventarios.

---
**Descripción del Proyecto:**
Este backend permite gestionar información del inventario de una distribuidora, incluyendo:

- Registro de productos
- Actualización de información
- Consulta de datos
- Eliminación de registros

El desarrollo se realizó por etapas:
1. Diseño de interfaz (Frontend)
2. Creación de la base de datos (MySQL)
3. Desarrollo del backend usando Java (CRUD)

---
**Mi Contribución Personal**
En esta parte del proyecto participé en:

- Implementación del CRUD en Java  
- Conexión con base de datos MySQL  
- Validaciones básicas para los datos del inventario  
- Estructura de paquetes y clases  
- Lógica para agregar, eliminar, actualizar y consultar productos  
- Pruebas funcionales del CRUD  
---
**Tecnologías Utilizadas**
- Java 
- MySQL
---
**Estructura del Backend**
Incluye:
- Clases para la gestión de productos  
- Clases para conexiones a base de datos  
- Métodos CRUD (Create, Read, Update, Delete)  
---
**Cómo Ejecutarlo**
1. Clonar o descargar este repositorio  
2. Configurar conexión MySQL en tu sistema  
3. Importar el proyecto en NetBeans, IntelliJ o Eclipse  
4. Ejecutar las clases principales del CRUD  
---
**Trabajo en Equipo**
Este proyecto fue desarrollado en grupo.  
Este repositorio refleja **mi aporte individual** dentro del módulo backend CRUD.
